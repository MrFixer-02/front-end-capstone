# CLAUDE.md

## Project Context

This is a setup repository for the Frontend AI Engineering track, separate from any cybersecurity portfolio work. Its only purpose is to prove that the frontend AI toolchain (Node.js LTS, Git, GitHub, and an AI coding assistant) is installed and working correctly. Future frontend capstone project work will build on top of this setup.

## Rules for Helping Me

- Keep changes scoped to what was asked. Do not add unrelated features, folders, or tooling.
- Before making a major change (new dependency, restructuring files, changing the stack), briefly explain the reasoning first.
- Prefer editing existing files over creating new ones.
- Ask before doing anything destructive (deleting files, force-pushing, rewriting history).

## Preferred Stack

- Node.js LTS
- Git + GitHub for version control
- Claude Code or Cursor as the AI assistant
- Keep additional frameworks/libraries minimal until a specific capstone project requires them

## Explain Reasoning

Before any major change, explain in plain terms:
- What is being changed
- Why it's needed
- Any tradeoffs

## Beginner-Friendly

I'm learning this workflow. Suggestions should be practical and beginner-friendly — avoid unnecessary jargon, and explain unfamiliar concepts briefly when they come up.

## Commit Style

Use [Conventional Commits](https://www.conventionalcommits.org/) for all commits (e.g. `docs:`, `chore:`, `feat:`, `fix:`). Keep commit messages short and descriptive.
